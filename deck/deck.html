仕組み THE SOLUTION

Physics you can prove.

Every ball's outcome is derived deterministically from a seed committed onchain before the session starts. Anyone can re-run the same physics and get the same result. Bit-identical. No trust required.

一
Commit the seed onchain.
Session starts → random seed posted to a Solana tx. Visible to anyone, tamper-proof, frozen forever.
二
Derive each ball by hash chain.
Ball n gets seed = SHA256(sessionSeed ‖ n). Launch velocity, jitter, and outcome all drop out of that hash.
三
Deterministic physics simulation.
Matter.js tuned for bit-identical trajectories across machines. Same seed → same bounce → same pocket.
四
Anyone verifies in 50ms.
Replay the physics locally from just the session tx + ball index. If the pocket matches, the house didn't cheat.
